## Roadmap execution guardrails

Refactoring work is executed one roadmap item at a time according to the
[CODEX execution plan](docs/architecture/CODEX-EXECUTION-PLAN.md), the
[master roadmap](docs/roadmap/master-refactoring-roadmap.md), and the phase files
under `docs/roadmap/`.

Before a roadmap commit is marked ready for review, use the reusable
[roadmap commit checklist template](docs/roadmap/templates/commit-checklist.md)
to confirm scope, validation, documentation updates, and PR readiness evidence.
